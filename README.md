### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、医生、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 居民健康信息
6. 居民就诊
7. 居民医保信息
8. 医生管理
9. 医生预约管理


**医生功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 药物信息管理
5. 居民健康信息管理
6. 居民就诊管理
7. 居民医保信息管理
8. 医生预约管理



**管理员功能：**
1. 登录
2. 个人中心(修改密码、个人信息）
3. 基础数据管理（药物剂型管理、科室管理、药物类型管理、医保类型管理、预约类型管理、职位管理）
4. 药物信息管理
5. 居民健康信息管理
6. 居民就诊管理
7. 居民医保信息管理
8. 医生管理、医生预约管理
9. 用户管理
